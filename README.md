Snake Ai


This project consists of building the classic snake game, and then teaching an AI to play the game.

Features

- Snake movement using keyboard controls

- Adjustable movement speed

- Game over detection

- Snake growth system

- Pygame graphics and rendering

Things learned throughout this project:

- How to use pygame

Skills used:

- Python 3

- Git/Github

- Pygame
